### User Personas

#### 1. **The Academic/Student** (Primary - ~60%)
- Uses Windows/Mac desktop
- Searches "text to latex" or "convert equation to latex"
- Writing papers, homework, thesis
- Needs: Math equations, formulas, scientific notation
- Pain point: Don't know LaTeX syntax
- High volume, medium complexity conversions

#### 2. **The Researcher/Professional** (Secondary - ~25%)
- Mac-heavy, uses Chrome
- Often comes direct (bookmarked)
- Writing academic papers, documentation
- Needs: Complex equations, multi-line formulas
- Power user - hits character limits
- Good upgrade candidate

#### 3. **The Developer/Technical Writer** (~10%)
- Linux/ChromeOS users, Firefox
- Writing documentation, README files
- Needs: Code blocks, inline math
- Lower frequency, quick conversions

#### 4. **The Casual User** (~5%)
- Mobile users
- One-time use, homework help
- Simple conversions
- Low retention
